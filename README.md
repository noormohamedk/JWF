Jasoor Welfare Foundation (JWF)
About the Project

The Jasoor Welfare Foundation (JWF) website is a responsive platform developed to present the foundation’s activities, initiatives, and community programs. It serves as a digital space to inform, engage, and connect with people interested in social welfare and community development.

The platform highlights various initiatives including educational support, awareness programs, youth development, healthcare activities, and relief efforts.

Features
Responsive design for mobile and desktop
Clean and modern user interface
Structured project and event pages
Image gallery for showcasing activities
Contact and engagement section
Social media integration
Optimized and lightweight performance
Tech Stack
HTML5
CSS3
JavaScript
Font Awesome

Purpose
The purpose of this project is to create a professional online presence for Jasoor Welfare Foundation, enabling better communication of its mission, activities, and impact. It also aims to encourage community participation and support.
